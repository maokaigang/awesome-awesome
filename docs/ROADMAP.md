# Roadmap

## Phase 1: Bootstrap

- Establish category structure and contribution rules.
- Add initial high-confidence seed repositories.
- Enable automated validation in CI.

## Phase 2: Growth

- Expand categories (cloud, database, testing, language-specific).
- Add maintainers for category ownership.
- Introduce monthly quality review issues.

## Phase 3: Maturity

- Add metadata per entry (last checked date, stars, activity tags).
- Build a generated index page from structured data.
- Publish periodic "newly added awesome lists" digest.
